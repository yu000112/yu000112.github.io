DBMS(数据库管理系统)–执行–> SQL(结构化查询语言)–操作–>DB(数据库)

  • SQL是一套标准，程序员主要学习的是SQL语句，这个SQL在mysql中可以用，在oracle中也可以用。

  • net start mysql(开启) net stop mysql(关闭) 必须用管理员身份运行cmd

    • 其他服务的启停都可以采用以上命令 net start/stop 服务名称

  • 端口号：端口号是任何一个软件/应用都会有的，端口号是应用的唯一代表。端口号通常和IP地址在一块，IP地址用来定位计算机，端口号port用来定位计算机上某个服务/应用的。在同一台计算机上，端口号不能重复，具有唯一性。

  • mysql数据库启动的时候，此服务占有的默认端口号是3306

  • 使用客户端登陆mysql数据库 mysql -uroot -p密码

    退出 exit

  • 查看mysql中有哪些数据库？ show databases;

    查看数据库下有哪些表？ show tables;

    查看表中的数据 select * from 表名; （效率低，可读性差）

    不看表中的数据，只看表的结构 desc 表名;

    选择使用数据库 use 数据库名字;

    查看数据库的版本号 select version();

    创建数据库 create database 数据库名字; (命令可以不区分大小写 )

    新建数据库–>use数据库–>导入表

  • 数据库的最基本单元是表：table ,因为表比较直观

    • 行（row）:被称为数据/数据
    • 列（column）:被称为字段

  • SQL 语句分类 DQL(select) DML(insert delete update) DDL(create drop alter) TCL (commit rollback) DCL(grant revoke)

SQL语法

1、查询 （select 不会进行修改操作，只负责查询）

  • 普通查询

select 字段名1,字段名2 from 表名; (如果是多个字段，则这样写；一个字段就写一个即可)

as 可以起别名，也可以省略as

select 1,2 22 from biaoming; (对as前面的那个名字起别名)

select dep ‘dep 1’ from biaoming;

字段可以参与运算

select ename,sal*12 as ‘yealsal’ from emp;

  • 条件查询

select

​ 字段1，字段2，字段3…

from

​ 表名

where

​ 条件;

mysql> select empno,ename,sal from emp where sal >=2450 and sal <= 3000;

等于select empno,ename,sal from emp where sal between 2450 and 3000;

数据库中不能用=衡量null，需要用 is null/is not null，因为数据库中地null代表什么也没有，不是一个值。

mysql> select empno,ename,job,sal from emp where job = ‘manager’ and sal > 2500;

and 优先级大于 or

**is null / is not null in / not in **

mysql> select empno,ename,job,sal from emp where job in(‘manager’, ‘salesman’);

等于select empno,ename,job,sal from emp where job = ‘manager’ or job = ‘salesman’;

找出名字中含有o的：

mysql> select ename from emp where ename like ‘%o%’;

找出名字以t结尾的：

mysql> select ename from emp where ename like ‘%t’;

找出名字以k开始的：

mysql> select ename from emp where ename like ‘k%’;

找出第二个字母是a的:

mysql> select ename from emp where ename like ‘_a%’;

找出名字中有“_”的：

select ename from emp where ename like ‘%\_%’;
